# Daily World News 🌍

自動化國際新聞整理系統。每天早上定時從全球主要新聞來源蒐集重點新聞，產出中文摘要。

## 功能

- 🌐 涵蓋全球主要國家/地區的新聞來源
- 📰 每日自動整理重點新聞
- 🇹🇼 中文摘要輸出
- 📡 支援 Telegram 推送

## 新聞來源

詳見 [SOURCES.md](SOURCES.md) — 歡迎討論調整！

## 每日新聞

產出的每日摘要存放在 `summaries/` 目錄下，格式：`YYYY-MM-DD.md`

## 調整來源

1. 編輯 `SOURCES.md` 新增/移除來源
2. 提 Issue 或 PR 討論
3. 來源會在下次執行時自動生效
