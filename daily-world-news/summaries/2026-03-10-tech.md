# 科技日報 — 2026年3月10日（週二）

## 🤖 AI／LLM

### OpenAI 收購 Promptfoo，強化企業 Agent 安全
OpenAI 宣布收購 AI 安全測試平台 Promptfoo，並將其技術整合至企業 Agent 平台 OpenAI Frontier。Promptfoo 已被超過 25% 的財富 500 強企業使用，專門提供 LLM 應用的自動化紅隊測試、提示注入偵測、資料洩漏防護等功能。收購後將持續維護其開源 CLI 專案，同時在 Frontier 中內建安全與合規能力。這顯示隨著 AI Agent 進入真實企業工作流程，安全評估已成為基礎需求。
🔗 https://openai.com/index/openai-to-acquire-promptfoo

### Claude 登頂美國 App Store，下載量暴增 240%
Anthropic 的 Claude 首次超越 ChatGPT 成為美國 Apple App Store 第一名免費 App。2 月下載量月增 240%（約 110 萬次增長），目前在美、加、德、愛爾蘭、盧森堡五個市場排名第一。Anthropic 的 Mike Krieger 表示每日新增用戶超過 100 萬。此變化源於 CEO Dario Amodei 公開拒絕配合五角大廈要求，而 OpenAI 同期因急簽國防部合約而遭到輿論反噬，Sam Altman 承認交易「倉促」且「觀感不佳」。
🔗 https://www.businessinsider.com/claude-number-1-app-stores-chatgpt-apple-google-ai-2026-3

### OpenAI 硬體負責人因五角大廈合約辭職
OpenAI 一名高層硬體主管在 3 月 7 日辭職，聲明中提到對「監控美國人」的擔憂。這是 OpenAI 與五角大廈合約引發的最新內部動盪，也反映出科技業在軍事 AI 合作議題上持續分裂。
🔗 https://glassalmanac.com/ai-has-an-important-role-sparks-resignations-in-march-2026-what-changes-now/

### Hackaday 探討：LLM 進入「生產力高原期」後的真正用途
Hackaday 發表長文討論 LLM 正在攀向 Gartner 炒作週期的「膨脹預期高峰」，預測當前的 AI 炒作將不可避免地崩潰，但真正有價值的應用將在「生產力高原期」浮現。文中引述一位擁有五十年經驗的軟體工程師試用 AI 編程工具後的反應，認為這類似於看到第一輛汽車的震撼。
🔗 https://hackaday.com/2026/03/09/ask-hackaday-what-will-an-llm-be-good-for-in-the-plateau-of-productivity/

### Cybersecurity 2026：企業 AI Agent 的安全挑戰
Netskope 的 Amit Datar 在專訪中指出 2026 年網路安全的關鍵趨勢：AI 驅動的攻擊面正在超越 GenAI 模型本身、Agentic AI 部署需要安全護欄、以及 MCP（Model Context Protocol）帶來的新風險。
🔗 https://www.cxodigitalpulse.com/cybersecurity-trends-2026-securing-ai-agents-in-the-enterprise/

## 🤖 Agent 框架

### Google Workspace CLI 為 AI Agent 打開大門
Google 推出 Workspace CLI，將 Gmail、Drive、Calendar API 打包成一個工具供 AI Agent 使用，並支援 OpenClaw 整合——但此工具並非 Google 官方支持。這讓 AI Agent 可以直接操作用戶的 Google Workspace 資源。
🔗 https://www.technology.org/2026/03/09/googles-new-cli-lets-ai-rummage-through-your-workspace-gmail-included/

### AI Storm 推出 ISEAI 雲端 OpenClaw 平台
AI Storm 推出 ISEAI 雲端服務，讓使用者無需配置 Docker 或雲端基礎設施即可直接在瀏覽器上運行 OpenClaw AI Agent。平台內建 Discord 整合，並提供五個預設 Agent（辦公自動化、內容生成、圖像創作、程式輔助、通用問答）。OpenClaw 在 GitHub 上已累積超過 3 萬顆星。
🔗 https://itbusinesstoday.com/tech/ai/ai-storm-launches-iseai-for-cloud-openclaw-ai-agents/

### Alibaba 開源 OpenSandbox：AI Agent 安全沙箱 API
阿里巴巴發布 OpenSandbox，為軟體開發者提供統一、安全、可擴展的 API，用於自主 AI Agent 的隔離執行環境。
🔗 https://www.marktechpost.com/2026/03/03/alibaba-releases-opensandbox-to-provide-software-developers-with-a-unified-secure-and-scalable-api-for-autonomous-ai-agent-execution/

### Google Cloud 發布自主網路運營新 Agent
Google Cloud 在 MWC 2026 上發布自主網路運營（Autonomous Network Operations）框架的新 Agent，目標讓電信服務商從手動管理過渡到零接觸運營。同時開源了電信資料管線和資料模型原始碼。
🔗 https://cloud.google.com/blog/topics/telecommunications/new-agents-for-the-autonomous-network-operations-framework

🔗 https://blogs.nvidia.com/blog/nvidia-agentic-ai-blueprints-telco-reasoning-models/

## 💻 基礎設施

### Oracle 的 AI 資料中心困境：OpenAI 不再擴大 Abilene 合作
CNBC 報導 OpenAI 不再計劃擴大與 Oracle 在德州 Abilene 的 Stargate 資料中心合作，原因是 AI 晶片迭代速度超過資料中心建設速度。Abilene 使用的是 NVIDIA Blackwell 處理器，但電力預計一年後才上線，屆時 OpenAI 希望已能使用下一代 Vera Rubin 晶片。NVIDIA 現在每年推出一代新晶片（而非過去的兩年），Vera Rubin 推理效能是 Blackwell 的五倍。這揭示了 AI 基礎設施投資的結構性風險。
🔗 https://www.cnbc.com/2026/03/09/oracle-is-building-yesterdays-data-centers-with-tomorrows-debt.html
