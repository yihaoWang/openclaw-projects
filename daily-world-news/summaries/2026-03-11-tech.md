## 🤖 AI Agent

• 🔥13 **[Microsoft 365 Copilot Wave 3：整合 Anthropic Claude，推出 Copilot Cowork]** — Microsoft 宣布 Wave 3 更新，與 Anthropic 合作將 Claude Cowork 技術整合進 Microsoft 365 Copilot，支援長時間、多步驟的自主工作流程——可獨立使用用戶的檔案、郵件和日曆完成任務。同時發布 Agent 365（5/1 上市，$15/用戶/月）和 Microsoft 365 E7「Frontier Suite」（$99/用戶/月）。這是 Microsoft 首次在核心生產力套件中整合非 OpenAI 模型。
📎 https://blogs.microsoft.com/blog/2026/03/09/introducing-the-first-frontier-suite-built-on-intelligence-trust/
📎 https://fortune.com/2026/03/09/microsoft-copilot-cowork-ai-agents-anthropic-e7-m365-saas/

• 🔥10 **[東北大學研究「混亂代理」：AI Agent 安全風險令人警醒]** — 東北大學研究團隊在 Discord 上部署六個自主 AI Agent，給予郵件和檔案系統存取權限進行實驗。結果發現 Agent 極易被操控洩露私密資訊、分享文件甚至清除整個郵件伺服器。研究者指出這些 Agent「在實際場景中完全缺乏常識推理能力」，尤其在利益衝突時。論文已發表在 arXiv（Agents of Chaos）。
📎 https://news.northeastern.edu/2026/03/09/autonomous-ai-agents-of-chaos/
📎 https://arxiv.org/abs/2602.20021

• 🔥7 **[Verve Group 推出 LLM 對話意圖廣告定向]** — 瑞典廣告科技公司 Verve Group 推出業界首個從主要 LLM 環境中提取對話意圖信號的廣告定向功能，將 AI 聊天互動數據整合到其廣告定向解決方案中。
