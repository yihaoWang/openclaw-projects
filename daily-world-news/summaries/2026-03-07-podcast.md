早安，今天是三月七號，星期五。

今天要跟你聊的事情份量很重。國際上，美國和以色列對伊朗的軍事行動進入第七天，Trump 直接喊出「無條件投降」，中東局勢不斷升級。美國那邊呢，二月的就業報告出了一個大冷門——居然是負增長。而科技圈這邊也很精彩，OpenAI 推出了 AI 資安代理，Anthropic 被 Pentagon 正式封殺，Claude Code 則登上了最受歡迎的 AI 編碼工具。東西很多，我們一個一個來。

先從中東講起。

美以聯軍對伊朗的攻擊已經進入第七天了。以色列軍方聲稱他們已經執行了超過兩千五百次空襲，摧毀了伊朗百分之八十的防空能力，現在基本上已經取得了「近乎完全的空中優勢」。德黑蘭和貝魯特一整夜都在被轟炸。根據半島電視台的數據，伊朗方面至少已經有一千三百多人喪生。

Trump 在這件事上的態度非常強硬，他直接說「不會有任何協議」，白宮發言人也表示，只有當伊朗「不再對美國構成威脅」的時候，Trump 才會認為伊朗算是投降了。白宮還暗示這場軍事行動可能會持續四到六週。

有幾個很值得關注的細節。第一個是俄羅斯的角色。有報導指出俄羅斯一直在向伊朗提供美軍艦艇和飛機的位置情報。這個消息如果屬實，代表這場衝突的背後有更大的地緣博弈在進行。第二個是伊朗總統 Pezeshkian 在社交媒體上暗示，已經有一些國家開始進行斡旋，但他沒有說具體是哪些國家。

英國那邊的反應也值得注意。英國駐巴林大使發了一段影片，宣布皇家空軍會派戰機飛越巴林上空，保衛他們所謂「全球最親密的盟友之一」。但別忘了，這週一才有一架無人機攻擊了塞浦路斯的英國空軍基地 Akrotiri。所以英國其實已經直接捲入了這場衝突。其他歐洲國家像德國、愛爾蘭、比利時、荷蘭，目前還是以外交手段回應為主。

這場戰爭最直接的經濟影響就是油價。布倫特原油已經飆破了九十美元一桶，美國的汽油價格也跟著漲。這個對全球通膨的壓力不容小覷。

好，把目光從中東轉到美國國內。

今天最震撼市場的消息就是美國二月的就業報告。非農就業人數居然減少了九萬兩千人。市場本來預期是增加的，結果出來的是負數。失業率也升到了百分之四點四。醫療保健業是跌幅最大的行業之一，部分原因是嚴冬天氣和一場大型醫療機構的罷工。

這份數據意味著什麼呢？它可能會逼迫聯準會重新考慮降息的時間點。但問題是，你一邊是就業在惡化，暗示經濟在放緩，另一邊是油價飆漲帶來的通膨壓力。這就把聯準會推到了一個很尷尬的位置——降息怕通膨，不降息怕經濟衰退。

華爾街的反應很直接。道瓊指數暴跌了五百六十二點，跌幅超過百分之一。標普五百和納斯達克也都下挫了百分之一左右。科技股、金融股和工業股是跌最多的板塊。

另一個美國內政的消息是，Trump 開除了國土安全部長 Kristi Noem。她上任以來一直面臨批評，現在終於被換掉了，Trump 也已經提名了繼任者。

再來一個有趣的新聞。任天堂起訴了美國政府，要求退還從二零二五年以來因為 Trump 的關稅而多繳的費用，還要加利息。這件事的背景是，最高法院已經裁定那些基於《國際緊急經濟權力法》徵收的關稅是違法的。任天堂表示這些關稅也是導致 Switch 2 預購延遲的原因之一。這應該只是第一張骨牌，接下來可能會有更多企業跟進提告。

接下來看看亞洲。

中國的全國人大正在開會，也就是所謂的「兩會」。李強在開幕式上宣布今年的 GDP 增長目標定在百分之四點五到五。這是數十年來中國設定的最低增長目標。背後的原因很清楚——通縮壓力和貿易戰的影響。值得注意的是 Trump 預計三月三十一號要訪問中國，所以這次兩會某種程度上也是在為那次會面做鋪墊。

一個很有意思的信號來自台海。根據路透社的報導，過去一整週，台灣周邊完全沒有偵測到中國空軍的活動。這是近年來的第一次。以前可是每天都有的。有分析認為這可能是北京在重新調整對台策略，也可能跟兩會期間的政治敏感性有關，又或者跟中東局勢分散了國際注意力有某種關聯。不管原因是什麼，這個變化本身就值得密切觀察。

日本今天有一個重磅消息。厚生勞動省正式批准了全球首個基於 iPS 細胞的再生醫療產品上市。iPS 細胞就是誘導多能幹細胞，這項技術的發明者山中伸彌在二零一二年獲得了諾貝爾獎。現在經過十幾年的研究，終於走到了商業化的這一步。這是再生醫學領域的一個里程碑。

日本的另一個關注點是能源安全。因為中東戰事導致的石油供應危機，日本正在考慮動用國家石油戰略儲備。日本首相高市早苗今天也跟加拿大總理 Mark Carney 會面，雙方同意建立網路安全和經濟安全的雙邊協商機制。

對了，體育迷可能已經知道了——世界棒球經典賽正在進行，大谷翔平在日本對中華台北的比賽中擊出了滿貫全壘打，日本大勝。

好，聊完國際局勢，我們來看看科技圈今天有什麼動靜。說實話，今天科技新聞的戲劇性一點不輸國際新聞。

最大的一條。Pentagon 正式把 Anthropic 列為了「供應鏈風險」。什麼意思呢？就是所有的國防承包商，從今以後都不能在跟 Pentagon 的合約裡使用 Claude 了。事情的起因是 Anthropic 要求 Pentagon 承諾不會用 Claude 來做國內大規模監控或者全自主武器。而 Pentagon 想要的是不受限制的存取權。雙方談不攏，結果 Trump 直接說「我們像開除狗一樣開除了他們」。

這件事有幾個層面值得思考。第一，Anthropic 是目前唯一一家在美國政府機密網路上部署過 AI 模型的公司。他們之前拿到了一份兩億美元的國防合同。所以被封殺的影響是很大的。第二，MIT Technology Review 今天發了一篇深度文章，問了一個根本性的問題——美國法律到底允不允許政府用 AI 對公民進行大規模監控？答案居然是「不清楚」。這件事的影響可能遠遠超出 Anthropic 本身。

Anthropic 的 CEO Dario Amodei 今天回應說，如果你仔細看 Pentagon 那封信的措辭，實際上禁令的範圍比初始報導說的要窄一些——只是禁止在「直接的國防合約」裡用 Claude。Amazon 也趕緊出來說，AWS 上的商業客戶用 Claude 完全不受影響。但不管怎麼說，這是科技公司和軍方之間圍繞 AI 倫理邊界的一次正面碰撞。

然後是 OpenAI。他們今天做了兩件事。第一件是發布了 GPT-5.4。對，你沒聽錯，兩天前才剛出 GPT-5.3，今天就來了 5.4。這種閃電式的發布節奏說明 OpenAI 正在拼命追趕。目前旗艦模型的競爭格局大概是 Claude Opus 4.6、Gemini 3.1 Pro 和 GPT-5.4 三強鼎立。

OpenAI 做的第二件事更有意思——他們推出了 Codex Security。這是一款專門用來偵測軟體安全漏洞的 AI 代理。以前叫 Aardvark，內部開發了一段時間，今天正式以 Research Preview 的形式上線。它能自動掃描你的程式碼庫，找出漏洞，然後提出修復方案。目前開放給 ChatGPT Enterprise、Business 和教育版的用戶，第一個月免費。這等於是 OpenAI 直接殺進了網路安全這個領域，對傳統的 SAST 和 DAST 工具形成了威脅。

另一個很振奮人心的消息來自 Pragmatic Engineer 的一份開發者調查。Claude Code 在上線僅九個月之後，就已經超越了 GitHub Copilot，成為開發者最常使用的 AI 編碼工具。同一份報告還提到 MCP 協議正在快速走向主流，NIST 也開始為 AI Agent 的安全制定標準。這個結果其實不太意外，如果你在 Twitter 或者 LocalLLaMA 社群上有關注開發者的討論，Claude Code 的口碑這半年來一直在瘋漲。

開發工具方面，TypeScript 6.0 的 Release Candidate 今天發布了。微軟說這是一個「獨特的版本」，暗示有比較大的架構變更。如果你是 TypeScript 開發者，值得關注一下。Ollama 也出了 0.17.7 版，新增了對 thinking levels 的支援。

晶片和硬體方面，Broadcom 的 CEO Hock Tan 放出了一個重磅預測——他們預計到二零二七年，AI 晶片的營收將「顯著超過一千億美元」。這個信號表明，儘管最近有些人擔心 AI 投資泡沫，但從硬體供應商的角度來看，需求不但沒有減緩，反而在加速。Broadcom 正在 NVIDIA 主導的市場裡快速搶份額，股價也因此漲了不少。

最後看一下市場。

美股三大指數今天全部收跌。道瓊跌了五百六十多點，標普跌了百分之一點一，納斯達克跌了百分之一。主要就是就業數據太難看，加上中東戰事推高油價，雙重打擊。

加密貨幣這邊也不太好。比特幣在這週稍早一度觸及七萬四千五百，但今天跌回了七萬以下，大概六萬九千八左右。美國現貨比特幣 ETF 在三月五號錄得了兩億兩千多萬美元的淨流出，終結了之前連續三天超過十一億的淨流入。以太坊和 Solana 相對比特幣表現好一些，分別在兩千一百七十和九十三左右。

另外加密圈還有一個爆雷的消息。BlockFills 這家加密借貸平台被爆出有七千五百萬美元的資金缺口。他們二月就已經暫停了提款，現在法院已經凍結了他們七十個比特幣的資產。投資者起訴稱他們挪用客戶資金來填補自己的營運虧損。這家公司背後的投資者包括 Susquehanna 和 CME Ventures，都是很大的名字。又是一個加密借貸平台出事的故事。

以上就是今天的國際新聞和科技動態整理。今天的訊息量確實很大——中東戰事在升級，美國經濟數據在惡化，科技巨頭在 AI 倫理和軍事用途上正面交鋒。這些事情看似各自獨立，但其實都在重塑我們所處的這個世界的權力結構和技術走向。

我們明天見。
