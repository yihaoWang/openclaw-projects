### OpenAI 發布 GPT-5.4：原生電腦操控、100萬 token 上下文、全新工具搜索系統
OpenAI 今日正式推出 GPT-5.4，號稱「最強大且最高效的前沿模型」。三個版本同步上線：標準版、GPT-5.4 Thinking（推理版）和 GPT-5.4 Pro（高效能版）。亮點包括：
- **100萬 token 上下文窗口** — OpenAI 史上最大
- **原生 Computer Use 模式** — 透過 API 和 Codex 讓 AI 像人類一樣操作電腦，跨應用程式工作
- **Token 效率大幅提升** — 部分任務節省高達 47% token 用量
- **Tool Search 系統** — 革新 API 工具調用方式，模型可按需查詢工具定義，大幅降低多工具場景成本
- **幻覺減少 33%**（與 GPT-5.2 比較），整體回覆錯誤率下降 18%
- **CoT 可控性安全評估** — 開源工具測試推理模型是否能刻意隱藏思維鏈，GPT-5.4 Thinking 版本表現良好
- 在 OSWorld-Verified、WebArena Verified、GDPval（83%）、Mercor APEX-Agents 等基準測試中刷新紀錄
- 來源：[TechCrunch](https://techcrunch.com/2026/03/05/openai-launches-gpt-5-4-with-pro-and-thinking-versions/) / [VentureBeat](https://venturebeat.com/technology/openai-launches-gpt-5-4-with-native-computer-use-mode-financial-plugins-for) / [The Verge](https://www.theverge.com/ai-artificial-intelligence/889926/openai-gpt-5-4-model-release-ai-agents)
