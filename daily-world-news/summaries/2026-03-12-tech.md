# 科技新聞摘要 — 2026年3月12日（JST）

---

## 🔥 重點頭條

### 1. NVIDIA 發布 Nemotron 3 Super — 為 Agentic AI 而生的開源大模型
NVIDIA 今日推出 **Nemotron 3 Super**，一款 1200 億參數的 MoE（Mixture of Experts）模型，實際運算只使用 120 億活躍參數。該模型專為多 Agent 工作流設計，擁有 **100 萬 token 的上下文窗口**，吞吐量比同級模型高出 5 倍。混合架構結合了 Mamba 和 Transformer，採用完全開放的訓練管線。Perplexity、CodeRabbit、Palantir、Siemens 等已宣布整合。這是 NVIDIA 在 GTC 2026（3/16-19 舉行）前的重磅先行發布。

**評分：9/10** — 開源 + 百萬上下文 + 高效推理，可能成為 Agent 領域的標桿模型。

### 2. NVIDIA 計劃推出「NemoClaw」開源 AI Agent 平台
根據 Wired 獨家報導，NVIDIA 正在開發名為 **NemoClaw** 的開源 AI Agent 平台，面向企業場景。已向 Salesforce、Cisco、Google、Adobe、CrowdStrike 等公司洽談合作。該平台將支持企業在安全環境中部署 AI Agent 執行內部任務。此舉標誌著 NVIDIA 從硬體公司向 AI 軟體生態系統的戰略轉型。

**評分：8/10** — NVIDIA 押注 Agentic AI 平台層，但仍處報導階段，尚未正式發布。

### 3. OpenAI 收購 Promptfoo 強化 AI Agent 安全
OpenAI 宣布收購開源 AI 安全測試平台 **Promptfoo**，目標加強 Agent 場景下的 prompt injection 防禦、工具執行安全及數據洩漏防護。Promptfoo 將繼續以開源方式運作。OpenAI 同時發布了關於設計 Agent 抵禦 prompt injection 的深度技術文章，指出現代攻擊已從簡單指令注入演化為社會工程式攻擊。

**評分：8/10** — Agent 安全成為行業焦點，收購 + 技術報告的組合非常及時。

### 4. Anthropic 起訴美國國防部 — AI 倫理紅線之戰升級
Anthropic 正式起訴 Trump 政府，抗議五角大廈將其列為「供應鏈風險」並要求軍方系統移除 Claude。起因是 Anthropic 拒絕解除 Claude 的使用限制（如禁止用於軍事殺傷決策）。**Microsoft 已提交法庭之友意見書支持 Anthropic**。內部國防部備忘錄已下令各軍事指揮官移除 Anthropic 技術。這場訴訟是 AI 倫理 vs 政府權力的標誌性案例。

**評分：9/10** — 極具歷史意義：科技公司為 AI 安全護欄對抗國家機器。

### 5. Rhoda AI 完成 $4.5 億 Series A 融資，估值 $17 億
機器人智能公司 **Rhoda AI** 脫離隱身模式，獲 Premji Invest 領投的 $4.5 億 Series A。其平台利用數百萬公開網路影片訓練模型，讓工業機器人在高度不確定的真實環境中自主操作。聚焦製造業和物流自動化。

**評分：7/10** — 物理 AI + 機器人是長期趨勢，但產品仍需驗證。
