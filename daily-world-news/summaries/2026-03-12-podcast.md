# Podcast Script — 2026年3月12日

大家好，這裡是每日新聞快報，今天是2026年3月12日，星期四。我是 Dan，帶你用十分鐘掌握今天最重要的世界大事和科技動態。

先從最令人關注的國際局勢開始。

美國和以色列對伊朗的軍事行動今天進入第12天，局勢出現重大升級。伊朗革命衛隊今天在霍爾木茲海峽對至少三艘商船開火，伊朗官方更公開警告，全球應做好油價飆升到每桶200美元的準備。美國國防部長 Hegseth 稱今天是「最密集打擊日」。

這場衝突的經濟衝擊正在全面展開。為了穩定油價，國際能源署協調了史上最大規模的戰略石油儲備釋放，32個國家將合計釋放約4億桶石油。日本首相高市早苗也宣布，日本將單方面釋放8000萬桶。布倫特原油在88到92美元之間劇烈震盪，美國汽油價格已連續11天上漲。

川普繼續發出矛盾信號。他先要求伊朗無條件投降，隨後又暗示戰爭可能在一兩週內結束。值得注意的是，CNN報導俄羅斯正在向伊朗提供無人機戰術建議，這為衝突增加了新的複雜性。

烏克蘭方面，澤連斯基指控匈牙利「強盜行為」，因為布達佩斯扣押了一批烏克蘭銀行運輸物資，包括價值8200萬美元的黃金和現金。俄軍對斯洛維揚斯克的空襲造成4人死亡、20人受傷。

亞太地區出現了一個有趣的變化。紐約時報報導，中國軍機在台灣周邊的活動突然大幅減少，過去一整週零架次。這打破了數年來幾乎每天都有的模式，分析師正在推測北京是否在重新調整對台策略。

接下來看金融市場。華爾街今天表現平淡，S&P 500微跌0.2%，道指下跌328點，主要受石油價格波動影響。美國二月CPI數據出爐，年增2.4%，完全符合預期，短期降息基本無望。

加密貨幣方面，比特幣在CPI數據後回落到約69,500美元。以太坊維持在2000美元以上。一個重要消息是以太坊基金會啟動了70,000枚ETH的質押計劃，使用Bitwise的基礎設施，這標誌著基金會從賣幣維運轉向質押收益的策略轉變。

Binance正面臨雙重壓力：美國司法部正在調查伊朗是否利用其平台逃避制裁，涉及金額近20億美元。Binance則反手起訴華爾街日報誹謗。

好，現在進入科技新聞的部分，今天有一個非常清晰的主題：Agentic AI 基礎設施大戰全面開打。

首先是 NVIDIA 的重磅發布。他們推出了 Nemotron 3 Super，這是一個1200億參數的開源MoE模型，但只有120億活躍參數。最大亮點是100萬token的上下文窗口，專門為多Agent工作流設計。Perplexity、CodeRabbit、Siemens 等已經宣布整合。這是 GTC 大會下週開幕前的前菜。

同時，根據 Wired 報導，NVIDIA 還在開發一個叫 NemoClaw 的開源 AI Agent 平台，已在跟 Salesforce、Google、Adobe 等巨頭洽談。NVIDIA 從賣 GPU 到建 Agent 平台，戰略轉型非常明確。

安全層面，OpenAI 收購了開源安全工具 Promptfoo，專門防禦 Agent 場景下的 prompt injection 攻擊。他們同時發布了一篇技術文章，指出現代攻擊已經從簡單的指令注入進化為社會工程式攻擊，傳統的「AI 防火牆」方式越來越難應對。

最具戲劇性的是 Anthropic 與美國政府的對峙。因為拒絕放寬 Claude 的軍事使用限制，五角大廈將 Anthropic 列為「供應鏈風險」，下令軍方移除其技術。Anthropic 正式起訴 Trump 政府，Microsoft 也提交了支持 Anthropic 的法庭意見書。這可能成為 AI 倫理領域最重要的法律先例之一。

最後，機器人AI公司 Rhoda AI 完成4.5億美元融資，估值17億美元。他們用網路影片訓練模型，讓工業機器人能在非結構化環境中自主操作。物理AI賽道持續升溫。

以上就是今天的重點。總結一下：伊朗衝突進入危險新階段，全球緊急釋放石油儲備；科技界則是 Agentic AI 基礎設施的全面軍備競賽，從模型、平台、安全到治理，所有層面同時推進。明天見！
